About WakandaDB

WakandaDB is an Open Source Model-driven NoSQL database engine with a comprehensive REST web interface. The database schema definition, server-side processing and querying are all done in JavaScript. Think of it as a JavaScript Relational Web Data Server... whah? ... keep reading.

REST Request

	__entityModel: "Country",

	companies: {
		__deferred: {
			uri: "/rest/Country(1)/companies?$expand=companies"
		}
