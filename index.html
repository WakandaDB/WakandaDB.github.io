Welcome to wakandaDB

WakandaDB is an open source object-relational database with a comprehensive RESTful web interface (create, read, update, delete and full indexed search). The database schema definition, server-side processing and querying are all done in JavaScript. Think of it as a JavaScript Relational Web Data Server... whah? ... keep reading.
